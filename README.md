# Analyseur-main-exodia
Projet Python personnel d'analyse et de simulation d'un deck Exodia.

Projet réalisé dans le cadre de mon apprentissage de Python.

Fonctionnalités :
- Simulation de 1000 mains de départ
- Comptage des monstres, magies et pièges
- Analyse du nombre de morceaux d'Exodia
- Système de score basé sur les rôles des cartes
- Identification de la meilleure et de la pire main
- Statistiques de qualité des mains
