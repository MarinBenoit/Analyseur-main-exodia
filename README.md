# Analyseur Exodia Yu-Gi-Oh!

## Description
```markdown
Projet personnel en Python simulant des milliers de mains d'ouverture d'un deck Exodia afin d'évaluer leur qualité statistique.
```
## Fonctionnalités

- Tirage aléatoire de mains
- Système de score Exodia basé sur les rôles des cartes
- Calcul de statistiques
- Recherche de la meilleure et de la pire main
- Analyse des probabilités d'obtenir des mains fortes
- Analyse du taux d'apparition de chaque carte sur un nombre de tirage donné (en chiffre et pourcentage)

## Technologies

- Python 3
- Module `random`
